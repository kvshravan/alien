# alien
Alien invader game made using python

# Installation steps
Step 1 -
```
git clone https://github.com/kvshravan/alien.git
```

Step 2 - 
```
cd alien
```
Step 3 - 
```
pip3 install pygame
```

Step 4 - 
```
python3 main.py
```
